# ตัวอย่างการแปลง Code เป็น สมการ

## ตัวอย่าง 1
```
for (i=0; i<n; i++){
    for (j=0; j<n; j++){
        sum += j;
    }
}
```
แปลงเป็นสมการได้เป็น
$$\sum_{i=0}^{n-1} \sum_{j=0}^{n-1}1 = \sum_{j=0}^{n-1}n = n^2 $$

```
for (i=1; i<n; i++){
    for (j=3; j<n-1; j++){
        sum += j;
    }
}
```
แปลงเป็นสมการได้เป็น
$$ \sum_{i=1}^{n-1} \sum_{j=3}^{n-2}1 = \sum_{j=1}^{n-1}(n-4) = (n-1)(n-4) = (n^2-5n+4) $$

> ไปอ่านเรื่องซัมเมชั่นเพิ่ม(ลำดับอนุกรม)
